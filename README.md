# PlantCare AI

PlantCare AI is a deep learning project that detects plant diseases from leaf images using transfer learning.

## Technologies Used
- Python
- TensorFlow
- CNN
- MobileNetV2
- Flask

## Features
- Upload plant leaf image
- Detect plant disease
- Display prediction result

## Dataset
New Plant Diseases Dataset (87K images)
